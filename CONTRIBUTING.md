# Contributing to Rubik's Cube algorithms

First off, thanks for taking the time to contribute! 

## How can you contribute?

### Share examples
- Feel free to send me nice examples of what you did.

### Reporting Bugs
- Use the GitHub [Issues] tab.
- Describe the steps to reproduce the bug.
- Attach your `data.json` if it's a configuration issue.
- Attach a screenshot of the dev console (Ctrl+Shift+I) or copy the relevant lines


### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests
3. Ensure the linter passes: `npm run lint`.
4. Submit a Pull Request with a clear description of the changes.

## Dev Setup

1. Clone the repository.
2. Install dependencies: `npm install`
3. Build in dev mode: `npm run dev`
